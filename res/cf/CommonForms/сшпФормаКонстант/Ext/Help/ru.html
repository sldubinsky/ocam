■ Флаг Использовать ESB – позволяет включить/выключить использование настроек ESB.

■ Поле Формат для сериализации – позволяет задать формат передаваемой информации.

■ Поле Длительность хранения (дней) – длительность хранения обработанных сообщений.

■ Поле Максимальное количество потоков обработки входящих – максимальное количество одновременных потоков обработки входящих сообщений.

■ Поле Максимальное количество потоков обработки исходящих – максимальное количество одновременных потоков обработки исходящих сообщений.

■ Поле Длительность ожидания (сек) – длительность ожидания обработчиком сообщений в очереди на обработку.

■ Флаг Отключить потоки обработки данных – предназначен для включения/выключения настроенных потоков обработки данных.
